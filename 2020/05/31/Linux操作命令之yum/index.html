Linux操作命令之yum

2020-05-31

修改yum源的下载地址

  1. 切换目录到cd /etc/yum.repos.d/
  2. 安装下载工具wget:yum install -y wget
  3. 下载阿里云的Centos-7.repo文件:wget -O CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
  4. 清理之前的缓存，并重新加载yum:yum clean all
  5. 建立一个缓存文件:yum makecache
  6. 查找软件，验证阿里云的yum源是否可以正常使用:yum search tomcat
注意：如果执行安装下载工具出现问题， 则需要到当前用户的根路径下执行 rm -f /var/run/yum.pid
